## Installation
Github:

    git clone https://github.com/I2rys/PortLol

NPM Packages:

    npm i request && npm i chalk
    
## Usage

    node index.js <url>

 - url - The website url to scan, make sure there is no / on the last part(e.g http://192.168.0.1).
